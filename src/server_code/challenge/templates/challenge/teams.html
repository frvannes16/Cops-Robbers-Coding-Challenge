Cops & Robbers



Guide

So it's easy to find, the primary command is python main.py [opposing team code]

If your are the Cops, then you have to find the robbers before they rob the banks. If you are the Robbers, then you are responsible for for robbing the banks without getting caught.

Code Setup

Step 1. - Sign up your team below.

Step 2. - Find your team type and download the respective code from the links above.

Step 3. - Unzip the files. Open controller.py and change the TEAM_CODE to your team code. controller.py is the only file that you will have to edit.

Step 4. - Run the program against somebody else's code. To do this, find another team that has a different role than you (COPS vs. ROBBERS). Note their team code, as you will be playing against each other. Use the list below to help.

Step 5. - Each team should run the following command: python main.py [other team's code]

Step 6. - A browser tab should open with your game in view. Now you're playing cops and robbers!
